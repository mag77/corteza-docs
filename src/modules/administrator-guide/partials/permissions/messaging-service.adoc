Access to messaging::
    Determines if the role is allowed to access {PRODUCT_NAME} {APP_NAME_MESSAGING}.

Grant permissions on messaging service::
    Determines if the role is allowed to grant permissions on {PRODUCT_NAME} {APP_NAME_MESSAGING}.

Access all settings::
    Determines if the role is allowed to access any {PRODUCT_NAME} {APP_NAME_MESSAGING} setting.

Manage all settings::
    Determines if the role is allowed to manage any {PRODUCT_NAME} {APP_NAME_MESSAGING} setting.

Create public channels::
    Determines if the role is allowed to create public {PRODUCT_NAME} {APP_NAME_MESSAGING} channels.

Create private channels::
    Determines if the role is allowed to create private {PRODUCT_NAME} {APP_NAME_MESSAGING} channels.

Create direct messages and group channels::
    Determines if the role is allowed to create direct {PRODUCT_NAME} {APP_NAME_MESSAGING} channels to a specific user or a set of.
